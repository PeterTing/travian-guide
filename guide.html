💡 基本設定

本攻略假設：伺服器速度x1、中立村鎮(無NPC或外交保護)、國王系統啟用。不同設定會有時間差異。

文化點數機制詳解

小型慶典

每個村莊每天自動產生CP(基於建築物數量)。小慶典獲得的CP = 該村當日CP產出(每次上限500，x1速度)；早期村莊每次約100-200 CP。慶典是主要CP來源。

市政廳Lv1 建造要求

  • 村莊大樓達到Lv10
  • 學院達到Lv10
  • 沒有其他市政廳(村莊內唯一)

慶典冷卻時間

市政廳Lv1時約24小時冷卻；升級TH會縮短冷卻時間，達成TH Lv20時可約每8小時舉辦一次。

到達2000 CP 計算

需要6~8場慶典(取決於建築數量和TH等級) + 被動CP累積(每天500上限)。快速建設更多建築可提高被動CP產出。

綠洲農業指南

綠洲內有動物守衛，英雄可率領低階兵種清理弱綠洲獲得戰利品。清早期資源補充關鍵。

逐小時建築順序（高盧人 — 最速路徑）

高盧人世界紀錄：76:41（約 3.2 天）。x1 速度伺服器，英雄點數全投資源。

第一階段：Hour 0-24（基礎建設）

Hour 0-2：

  • 完成新手任務（免費資源各 ~500）
  • 所有資源田 → Lv2（優先穀田）
  • 主府 → Lv3
  • 英雄：點數全投資源採集

Hour 2-12：

  • 主府 → Lv5
  • 資源田 → Lv3-4（重點穀田 Lv4）
  • 倉庫 Lv3、糧倉 Lv3
  • 地下室 Lv3（高盧 1.5× = 保護 450/種）
  • 英雄初次出征：清鐵/磚綠洲 → ~2,400 資源

Hour 12-24：

  • 資源田 → Lv5
  • 主府 → Lv7-8
  • 倉庫 Lv5、糧倉 Lv5
  • 英雄每 2-3 小時循環清怪（每穀消耗 = 各 40 資源獎勵）
  • Rally Point Lv1（若未蓋）

第二階段：Hour 24-48（衝 MB10 + Academy）

Hour 24-36：

  • ★ 主府 → Lv10（最關鍵）
  • 資源田 → Lv6
  • 倉庫 Lv6、糧倉 Lv6
  • 使館 Lv1（加入聯盟）

Hour 36-48：

  • ★ Academy → Lv1 → 持續升到 Lv10
  • 兵營 Lv1-3 → 訓練 5 Phalanx
  • 資源田 → Lv6-7

第三階段：Hour 48-72（Town Hall + Residence）

Hour 48-60：

  • Academy Lv10 完成 ✓
  • ★ 市政廳 Lv1 → 立即辦第 1 場慶典（~150-200 CP）
  • 居所 → Lv1 → 開始升級

Hour 60-72：

  • 居所 → Lv5-6
  • 資源田 → Lv7-8
  • 倉庫 Lv7-8、糧倉 Lv7-8（Settler 單隻 18,100！）
  • 第 2 場慶典（~200 CP）

第四階段：Hour 72-120（Residence Lv10 + Settlers）

Hour 72-96：

  • ★★★ 居所 → Lv10（解鎖 Settlers）
  • 第 3 場慶典
  • 倉庫 Lv9-10（容量 ≥ 18,000）
  • 資源田 → Lv8-9

Hour 96-120：

  • Academy 研究 Settler 技術
  • 訓練 Settlers ×3（18,100×3 = 54,300，高盧 1.7 小時/隻）
  • 第 4-5 場慶典
  • 英雄持續農綠洲補資源

第五階段：Hour 120-144（開村！）

  • 確認 CP ≥ 2,000 ✓
  • 選址：≤ 20 格、優先 15c、附近有穀綠洲
  • 發送 3 Settlers 開村 🎉
羅馬人時間線差異
  • 雙建築佇列：Day 1-2 同時升資源田 + 主府，快 30-40%
  • 英雄 +100/點（他族 +80），清怪更強
  • Settler 慢 25%（每隻 ~2.1 小時），建築優勢補回
  • 預估開村：Day 5-6

🛡️ 羅馬人完整攻略 (Day 5-6)

🎯 核心優勢

雙建築佇列 - 同時升級資源田和其他建築，加快前期發展速度。英雄戰鬥力強(+100/點)，綠洲清理效率高。

⚙️ 英雄配置策略

所有屬性點投入戰鬥力(Fighting Strength, +100攻擊力/點)。成為綠洲清理的最強戰士，快速積累資源。Day 1-2開始英雄農業補充資源。

🎊 慶典計劃

市政廳Lv1後立即開始，每24小時舉辦一次慶典。早期每場約100-200 CP，需6-8場才能累積2000 CP。時間約Day 2-6。

⚔️ 兵種訓練

Day 1-2: 訓練5-10個古羅馬步兵用於英雄護衛。無需大規模駐軍。綠洲戰利品販售補充維持費。

📅 完整4階段時間線

階段1: 基礎建設 (Hour 0-24)

Hour 0-6:

  • 完成新手任務獲取免費資源(木材、黏土、鐵礦各數百)
  • 所有資源田升級至Lv2-3 (優先穀田確保兵種維持)
  • 村莊大樓升至Lv3
  • 地下室Lv1 (保護100資源/類型)

Hour 6-12:

  • 村莊大樓升至Lv5 [雙佇列: 同時升級穀田至Lv4]
  • 其他資源田升至Lv4
  • 倉庫Lv3、糧倉Lv3
  • 準備訓練5個古羅馬步兵(共2,300資源)

Hour 12-24:

  • ⭐ 村莊大樓升至Lv8-10 (關鍵：最高優先級)
  • 資源田升至Lv5-6
  • 倉庫/糧倉升至Lv5
  • 兵營Lv1-3，訓練5個古羅馬步兵
  • 英雄開始清理攻擊力<50的綠洲(鼠、蜘蛛、蛇)
階段2: 學院衝刺 (Hour 24-48)

Hour 24-36:

  • 學院升至Lv10 [雙佇列: 同時升級資源田至Lv7]
  • 大使館Lv1 (無需升級，僅為今後使用)
  • 英雄持續農業，販售所有綠洲戰利品補充資源

Hour 36-48:

  • 市政廳Lv1 (需MB10+Academy10) → 立即啟動第1場慶典!
  • 居所開始建築 [0/等級]
  • 資源田升至Lv7-8
  • 倉庫/糧倉升至Lv7
階段3: 居所+CP推進 (Hour 48-96)

Hour 48-72:

  • 居所升至Lv10 (遊戲最耗時建築，預計36-48小時)
  • 第2、3場慶典已完成/排隊中
  • 資源田升至Lv8-9
  • 倉庫/糧倉升至Lv9-10
  • 英雄農業持續，庫存補充定居者訓練資源

Hour 72-96:

  • 研究定居者科技 (學院內，需Academy Lv5+)
  • 開始訓練3個定居者 (57,000資源: 木13,800+黏12,600+鐵17,400+穀13,200)
  • 第4、5場慶典啟動
  • 英雄庫存用於填補資源缺口 (英雄負載:4,000資源)
  • 追蹤CP計數 (目標: 72小時內2000+ CP)
階段4: 定居 (Hour 96-144)

Hour 96-120:

  • 驗證CP ≥ 2000
  • 3個定居者訓練完成 (預計Hour 80-96)
  • 選擇定居點: 20格內15綠洲或9綠洲村莊 (避免玩家定居點)
  • 準備軍隊護衛 (5個古羅馬步兵足夠，定居點通常無防守)

Hour 120-144:

  • 發送3個定居者至目標村莊
  • 定居完成 → 第2村莊建立! (Day 5-6完成)
  • 新村莊起始: 建造資源田、倉庫、糧倉 (接續主村莊發展)

🌲 高盧人完整攻略 (Day 4-5)★ 最快

🎯 核心優勢

最便宜的定居者(54,300資源) + 定居者訓練速度快25% = 最快達成第2村莊。強大的防禦能力(高盧方陣兵)讓綠洲清理更安全。

⚙️ 英雄配置策略

所有點數投入資源採集(Resources, +100資源/點)。高盧英雄戰鬥力弱，依靠方陣兵護衛農業，但資源採集速度超快補償農業效率。

🎊 慶典計劃

同羅馬人策略，TH Lv1後盡快開始，早期每場約100-200 CP，需6-8場慶典達2000 CP。時間約Day 1-5。

⚔️ 兵種訓練

Day 1-2: 訓練5-10個方陣兵(步兵防禦強)護衛英雄。Day 3後加訓雷神戰士(速度19)用於快速農業。高盧兵無需大規模駐軍。

⚠️ 高盧人特殊機制

無雙建築佇列 - 高盧人無法同時升級兩個建築。需要優先規劃建造順序以避免浪費時間。但定居者速度優勢彌補此缺陷。

📅 快速時間線摘要 (Day 4-5達成)

Day 0: 初期建設(資源田Lv5、倉庫Lv3)、英雄農業開始; Day 1-2: 主府升Lv10、學院衝Lv10、市政廳建立並啟動慶典; Day 2-5: 居所升Lv10期間持續慶典(需6-8場)、定居者訓練; Day 5-6: 定居者訓練完成、確認CP≥2000、發送定居者完成第2村莊

最後提示

⚠️ 常見陷阱
  • 不要忘記升級倉庫/糧倉 - 存儲容量不足會浪費資源
  • 居所升級是最耗時的 - 提早規劃資源以避免資源短缺
  • 英雄農業很關鍵 - Day 1開始就要充分利用英雄清理綠洲補充資源
  • 定居點選擇重要 - 避免玩家已定居的位置，優先選15綠洲村莊
💡 成功檢查清單
  • Hour 24: 主府Lv10 ✓ 學院開始/完成Lv8+ ✓
  • Hour 36: 市政廳Lv1 ✓ 第1場慶典啟動 ✓
  • Hour 72: 居所Lv10完成 ✓ 慶典持續進行中(目標6-8場) ✓ 定居者訓練開始 ✓
  • Hour 96: CP≥2000確認 ✓ 定居者訓練完成或即將完成 ✓
  • Hour 120-144: 第2村莊定居完成! ✓
