資源田 ROI 計算器

ROI（Return on Investment）告訴你「升一級資源田，幾天才能把投入的資源賺回來」。ROI 天數越短代表效率越高；一般來說同等級 ROI 下應優先升 ROI 最短的那種田。此計算器使用 Lumi 的慣例（L1 比對 0 產出，L2+ 比對前一級產出，疊加為加法 1 + 加成建築% + 綠洲% + 金幣%）。

ROI tells you "how many days of extra production recovers the cost of this field upgrade". Lower ROI = more efficient; given equivalent levels you should prioritize the lowest-ROI field first. This calculator follows the Lumi convention (L1 compared to 0, L2+ compared to previous level, additive stacking: 1 + bonus-building% + oasis% + gold%).

被動 CP 計算器

文化點（Culture Points）是唯一不會被偷走的「資源」，用於開新村。被動 CP = 村內建築等級的 CP 產出總和，公式 CP(L) = round(base × 1.2^L)。Lumi 強烈建議每個村至少做到 MB20 / 市場20 / 學院20 / 大使館20 / 市政廳10（約 529 CP/天，不計其他建築）。慶典會補一半以上的 CP；此計算只看被動部分。

Culture Points are the one "resource" that can't be stolen, and they gate village expansion. Passive CP = sum of each building's CP output, where CP(L) = round(base × 1.2^L). Lumi strongly recommends at least MB20 / Market20 / Academy20 / Embassy20 / Town Hall 10 in every village (~529 CP/day before other buildings). Celebrations add the rest; this calculator only counts passive CP.

    各遊戲階段推薦工具

    初期階段（前 2 週）

    • Culture Points - 計算何時能開二村、三村

    • Path Calculator - 計算掠奪來回時間

    • Search Inactives - 開始建立農場清單

    • Statistics - 監控周邊威脅

    Early Game (First 2 Weeks)

    • Culture Points - Calculate when to settle 2nd and 3rd villages

    • Path Calculator - Calculate raid round-trip time

    • Search Inactives - Start building farm lists

    • Statistics - Monitor surrounding threats

    中期階段（2-4 週）

    • Technology Calculator - 優化科技升級順序

    • NPC Calculator - 高效交換資源

    • GetterMap 2.1 - 分析區域聯盟分布

    • Battle Reports - 記錄和分析戰鬥

    Mid Game (Weeks 2-4)

    • Technology Calculator - Optimize tech upgrade order

    • NPC Calculator - Efficiently exchange resources

    • GetterMap 2.1 - Analyze regional alliance distribution

    • Battle Reports - Record and analyze battles

    後期階段（4 週+）

    • Trooptool - 聯盟層級的兵力管理和攻擊計畫

    • Defense Tables - 協調聯盟防禦

    • Crop Tool - 管理大軍糧食

    • Interception Calculator - 規劃高級防禦戰術

    • Path-Speed-TS - 分析對方情報

    • Save Troops - 在危機時刻保護部隊

    Late Game (Week 4+)

    • Trooptool - Alliance-level troop management and attack planning

    • Defense Tables - Coordinate alliance defense

    • Crop Tool - Manage large army crop supply

    • Interception Calculator - Plan advanced defensive tactics

    • Path-Speed-TS - Analyze enemy intelligence

    • Save Troops - Protect troops during crises
